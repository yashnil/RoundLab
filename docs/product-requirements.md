# Product Requirements

## MVP Goal
Build Dissio V1: an AI flow coach for Public Forum debaters.

## Core User Flow
1. User records or uploads a PF speech.
2. User selects speech type: Constructive, Rebuttal, Summary, Final Focus, or Crossfire.
3. App transcribes the speech.
4. App extracts arguments into claim/warrant/evidence/impact.
5. App generates a flow-style table.
6. App gives judge-specific ballot feedback.
7. App identifies top weaknesses.
8. App generates 3 personalized drills.
9. User can re-record and track improvement.

## Must-Have Pages
- Landing page
- Login/signup
- Dashboard
- Record/upload speech
- Transcript page
- Feedback report
- Flow table
- Drill practice page
- Progress dashboard

## Must-Have MVP Features
- Auth
- Audio recording/upload
- Speech metadata form
- Transcription
- Argument extraction
- Flow table
- Debate rubric scoring
- Ballot-style feedback
- Personalized drills
- Speech history
